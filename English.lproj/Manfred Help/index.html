Manfred Help (v 0.4.1)

What is Manfred?

Manfred is a Cocoa/Objective C based framework for inspecting (and eventually modifying) resources for the game Neverwinter Nights by Bioware Corp. (ERF/GFF based resources, at least). The primary focus, at least initially, is to provide Mac users an easy way to view and document resources for the OKDocs project. Manfred is an affiliated project of the Open Knights consortium.

Why Manfred?

I wanted to call it GNOME for General Neverwinter Object and Module Editor, but apparently this is locked up by some obscure unix desktop program. And Mighty Manfred was the name of my only gnome PC in my decades of Dungeons and Dragons. He was the sidekick of an illusionist named Tom Terrific. Bonus points to anyone who understands this relationship.

What can you do with Manfred?

At present you can open an inspector on a variety of NWN data files, using the File:Open menu item (or CMD-O). This currently includes modules (.mod, .nwn, and .sav) as well as other ERF and GFF format files, in particular hak files (.hak). Opening a module may take a while if it's particularly large.

For ERF format resources, like modules, the inspector will display the contents of the module as a list of resources with their type and codes. You can open an inspector on some resources by selecting the resources in the list, and then selecting Module:Inspect. At present GFF format resources (which covers the vast majority) and NSS (scripts) are covered.

Please note that the display of the resource contents is far from nice, and that there is at present no way to modify and save any resource contents. GFF resources may be dumped to an HTML-friendly format file, by pressing the button on the inspector window.

You can monitor recent improvements and updates to Manfred through the Release Notes

That's it? That's not very much.

Well, Manfred was a project mostly to allow me to learn more about implementing something in Objective C. As such there are a lot of lose ends. I intend to take it to the point where it can at least save simple changes to GFF resources, and maybe directly open them from files (and from .HAK and ERF files). Additional capabilities may be added at a later time, if there is interest.

Note that there are other projects at the Open Knights which may offer better (cross platform) tools. People are especially directed to the "neveredit" project.
