Manfred Release Notes

0.5 (2003 August 1)

  • Cleaned up Inspector window layouts for better behavior under window resizing (Bug 766319).
  • Enabled access to basic help from the Help menu item (Bug 766580).
  • Fixed inclusion of current version number and file name in file Get Info. (Bug 772054)
  • Some placeholder nib files added to be used in later releases.

0.4 (2003 July 7)

  • Added resource name/type as instance variable identity (so it knows its own name)
  • Show identity of property resource in HTML Dump. Make Dump to file (note it is only partial HTML, missing the prologue and close).
  • Placeholder ResourceInspector class to act as root for all resource inspectors.
  • Generalized the InspectorDocument to handle non-Module files for opening, in particular: .hak, .erf., and .sav.
  • Able to display .nss (NeverwinterScript Source) resources.

0.3 (2003 May 26)

  • Initial Dump to HTML. Changed Aggregate Type code from hex to decimal.
  • Removed null chars from ElementProperty names (fixing HTML dumps).
  • Labeled AggregateResource sub-inspector windows with file name.

0.2 (2003 May 5)

  • Able to unpack PropertyResource (GFF) data including all property types (Though not all have been debugged).
  • Able to inspect GFF resources via Module:Inspect when the resource is selected within the ERF inspector.

0.1 (2003 April 15)

  • Basic Document based app functionality: Module window open.
  • ERF (AggregateResource) Inspector accessed through Module:Inspect.
  • Module (TypedResource) Inspector access file type and version fields.
  • Aggregate Resource unpacks contents into header and creates member resources.
